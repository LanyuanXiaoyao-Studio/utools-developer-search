# 资源搜索 Lite
> 原插件「开发者搜索」更名为「资源搜索 Lite」, 其实际上为 [资源搜索](https://yuanliao.info/d/1875-0-0-3) 插件的简化版, 后续将不局限于开发者仓库搜索

在模板插件里直接搜索开发者仓库资源, 目前支持
- Maven
- Maven (Sonatype)
- npm
- Github
- Gitee(码云)
- Docker Hub
- PyPI
- RubyGems
- pub
- pub (CN)
- Nuget
- cartes.io
- LuaRocks
- Packagist
- Gradle Plugins

# 使用
`enter(回车)` 搜索/选择
`Ctrl + t` 打开设置
