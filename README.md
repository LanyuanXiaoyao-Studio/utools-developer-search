# 开发者搜索
在模板插件里直接搜索开发者仓库资源, 目前支持
- Maven
- Maven (Sonatype)
- npm
- Github
- Gitee(码云)
- Docker Hub
- PyPI
- RubyGems
- pub
- pub (CN)
- Nuget
- cartes.io
- LuaRocks

# 使用
`enter(回车)` 搜索/选择
`Ctrl + t` 打开设置
